# 📈 Stock Data Analysis & Visualization  

📊 Analyzing and visualizing stock price and revenue data for Tesla and GameStop  

## 📌 Overview  
This project focuses on extracting, analyzing, and visualizing stock market data for companies like Tesla and GameStop.  

It combines financial data such as stock prices and quarterly revenue to identify trends and insights.

---

## 🚀 Features  

- 📊 Extract stock data using Python  
- 📈 Visualize stock price trends  
- 💰 Analyze quarterly revenue data  
- 🔍 Compare stock performance with revenue growth  
- 📉 Generate insights using data visualization  

---

## 🛠️ Tech Stack  

- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## 📂 Files Included  

- `tesla_stock.ipynb` → Tesla stock analysis  
- `tesla_quarterly_revenue.ipynb` → Tesla revenue data  
- `gamestop_stock.ipynb` → GameStop stock analysis  
- `GME_quarterly_revenue.ipynb` → GameStop revenue  
- `stock_data_extraction.ipynb` → Data extraction scripts  

---

## ⚙️ How It Works  

1. Extract stock data  
2. Clean and process data using Pandas  
3. Analyze trends over time  
4. Visualize results using graphs  
5. Compare stock prices with revenue  

---

## 🎯 Objective  

To understand stock market behavior and visualize financial trends using Python and data analysis techniques.
